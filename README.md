# Wassalni

**Wassalni**: Your Journey, Connected.  
Effortlessly match rides, plan routes, and connect travelers—anywhere, anytime.

Wassalni is a next-generation mobility platform that bridges the gap between drivers and passengers using smart, real-time ride-matching. Built with cutting-edge web (React), mobile (Flutter/Dart), and robust backend technologies, Wassalni transforms the way you move. Whether you’re commuting across the city or exploring new destinations, Wassalni delivers a seamless, social, and secure travel experience—powered by intelligent algorithms and community trust.

- 🚗 Instant ride-matching & route optimization  
- 🌍 Multi-platform: Web & Mobile  
- ⚡ Real-time updates & notifications  
- 🔒 Safe, community-driven, and customizable  

Join the movement and redefine your journey with Wassalni!
